# Diabetic Retinopathy Prediction

A Machine Learning-based web application that predicts the likelihood of **Diabetic Retinopathy** using clinical parameters such as age, blood pressure, and cholesterol levels. The application is built using **XGBoost** for prediction and **Streamlit** for an interactive and user-friendly interface.

---

## 📌 Project Overview

Diabetic Retinopathy is one of the most common complications of diabetes and can lead to severe vision impairment if not detected early. This project aims to assist in early risk assessment by using machine learning techniques to analyze patient clinical data and predict the likelihood of diabetic retinopathy.

The model uses the following clinical parameters:

- Age
- Systolic Blood Pressure
- Diastolic Blood Pressure
- Cholesterol Level

Based on these values, the application predicts whether diabetic retinopathy is likely or not.
